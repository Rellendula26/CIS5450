# CIS5450 Big Data Analytics Projectss
End-to-end data science workflows including SQL-based data extraction, feature engineering, and machine learning models on large-scale datasets. 

Homework 1: This project builds an end-to-end machine learning pipeline to predict housing prices from structured real estate data, processing thousands of data points through feature engineering and regression models to achieve high predictive accuracy and quantify the impact of key features on price estimation.
Homework 2: This project builds a scalable data processing and machine learning pipeline on large-scale structured datasets, transforming raw data through SQL-based extraction, feature engineering, and model-ready preprocessing to enable efficient downstream analytics and predictive modeling.
